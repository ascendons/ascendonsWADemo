# Divisha Appointment System - Quick Reference Guide

## For Patients: WhatsApp Booking

### Regular Booking (3 Steps)

**Step 1: Start**
- Open WhatsApp → Send **"Hi"** to Divisha number

**Step 2: Register (First Time Only)**
- Provide: Name, Age, Gender
- Save your Patient ID

**Step 3: Book Appointment**
- Select: Doctor → Location → Date → Time
- Confirm booking
- **Save your Booking ID!**

### Reschedule Appointment
1. Send **"Hi"** → Select **"CANCEL/RESCHEDULE"** → **"RESCHEDULE"**
2. Tap on appointment to reschedule
3. Select new: Doctor → Location → Date → Time
4. Confirm new appointment

### Cancel Appointment
1. Send **"Hi"** → Select **"CANCEL/RESCHEDULE"** → **"CANCEL"**
2. Tap on appointment to cancel
3. Receive confirmation

### Walk-In / Emergency
1. Send **"Hi"** → Select **"WALK-IN/EMERGENCY"**
2. Provide details (if new patient)
3. Select: Location → Doctor → Date
4. Confirm walk-in appointment

## For Administrators: Common Tasks

### Add a Doctor
**Settings** → Create New User → Fill form → Role: DOCTOR → Create

### Set Doctor Schedule
**Settings** → Find Doctor → Set:
- Daily hours (e.g., 9 AM - 5 PM)
- Available days (check boxes)
- Unavailable dates (add specific dates)
- Click **Save**

### View Appointments
**Dashboard** → Select date → View list
- Use search bar to find specific appointments
- Use filters (Confirmed, Waitlisted, Walk-in, etc.)

### Manage Waitlisted Appointments
**Dashboard** → Click **"Waitlisted"** filter → View waitlisted patients
- When slot opens: Click appointment → **Reschedule** → Assign to new slot
- Contact patient to offer available slot

### Reschedule Appointment
Click appointment → **Reschedule** → Select new date/time → Confirm

### Cancel Appointment
Click appointment → **Cancel** → Confirm

### Check In Patient
Click appointment → **Check In**

### View Walk-In Appointments
**Dashboard** → Click **"Walk-in"** filter → View walk-in appointments
- Walk-ins don't have specific time slots
- Check in when patient arrives

### Add Patient Manually
**Patients** → **New Patient** → Fill form → Create

### Add Location
**Settings** → Locations → Add Location → Fill name → Add

## Appointment Statuses

| Status | Meaning |
|--------|---------|
| **Confirmed** | Appointment is booked ✓ |
| **Waitlisted** | Time slot full, on waitlist |
| **Completed** | Appointment finished |
| **Cancelled** | Appointment cancelled |
| **Walk-in** | Patient arrived without booking |

## Important IDs to Save

- **Patient ID**: Given after first registration
- **Booking ID**: Given after booking appointment (format: BID-1234567890)

## Quick Troubleshooting

### Patient Can't Book
- Check if doctors are added in Settings
- Check if doctor has schedule set
- Check if locations are added

### Appointment Not Showing
- Check date picker (might be on wrong date)
- Use search bar to find by name/phone
- Check filters (might be filtering it out)

### Doctor Not Available
- Check doctor's schedule in Settings
- Verify available days are checked
- Check for unavailable dates blocking the date
